# Portfolio Media Assets

This folder contains media assets for the Portfolio website.

## Structure

```
media/
├── images/     # Project screenshots, profile photos, backgrounds
├── icons/      # Custom SVG icons, favicons
└── docs/       # PDFs, documents (resume, certifications)
```

## Usage in HTML

```html
<!-- Images -->
<img src="media/images/aios-architecture.png" alt="AIOS Architecture">

<!-- Icons -->
<link rel="icon" href="media/icons/favicon.svg">

<!-- Documents -->
<a href="media/docs/resume.pdf">Download Resume</a>
```
